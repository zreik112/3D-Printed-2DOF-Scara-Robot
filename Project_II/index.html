Perspective view of the Robot

Here is a blueprint showing the overall robot structure.
As we can see the Robot is composed of 2 joints where each one is 100mm in length. So in total the robot has a full reach of 200mm with a rated payload of about a dozen of grams.

How to control the Robot

We have two ways to control the Robot: using the Keypad (X and Y buttons) or Manually via the web (a mesh that enables you to choose the point to go to). interface
