# 3D-Printed-2DOF-Scara-Robot
This repository contains all of the files needed in order to print, assemble and move the robot via the web UI. 

# List of materials 
2x Nema 17s 0.4Nm, 2x 6003RS Bearings, 2x TB6600 stepper drivers, 1x 24V power Supply, 
1x Arduino Uno, 
M4 Bolts.

# Note
The fodler named RobotScaraWebProject is the Arduino code developped using PlateformIO.<br>
Meanwhile the folder Project_II is the folder containing the website files.
